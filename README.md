# textlint-plugin-sql

text support for [textlint](https://github.com/textlint/textlint "textlint").

## Installation

    npm install textlint-plugin-sql

## Usage

Manually add sql plugin to do following:

```
{
    "plugins": [
        "sql"
    ]
}
```

## Tests

    npm test
